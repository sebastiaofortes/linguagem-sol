Linguagem SOL

Uma linguagem de programação para brasileiros desenvolvida por brasileiros!

O que é:

A linguagem SOL é uma linguagem de programação baseada em portugol e em C++ desenvolvida por Sebastião Fortes Avelar de Almeida com o objetivo de permitir que pessoas de fora do mundo da programação possam criar pequenos programas por meio de um código simples e de fácil compreensão, essa linguagem também tem o objetivo de facilitar o ensino de programação para iniciantes.

História

Como ela foi criada:

Seu compilador foi desenvolvido baseando-se no compilador Tiny C, um compilador de linguagem C open source que apresenta bom desempenho e estabilidade.

Atualmente é possível compilar código escrito em C juntamente com o código escrito em SOL por meio de uma versão estendida do compilasdor Tiny C.
